Forum Guides

Project Forum Guide

This is the place to showcase your game development work, prototypes, and creative ideas. **Allowed posts** include work-in-progress, announcements for your games, collaboration requests, and devlogs. **Not allowed** are off-topic content, low-effort or spam posts, and advertising unrelated servers.

Script Forum Guide

Only post working scripts with a clear description of what they do, how to use them, and any dependencies. No malicious code or copyright infringement. Encourage collaboration by making scripts open-source if possible.

Support Forum Guide

Make sure you tag your post correctly. Use the ticket system for complex issues. When reporting a bug, send your code as a code block, not a screenshot, and provide the exact error you're facing. Do not post in the #general channel.

Commissions Rules

Be honest about your skills and rates. All important discussions about a project (briefing, payments, modifications) should take place in a thread. Never accept or request money outside of secure, traceable methods. It is recommended to ask for an upfront deposit (typically 25-50%), but never pay the full amount before the final work is delivered.
